# interview-cooking-web
This project is for interview-cooking-web

# 1. Requirement
NodeJs v16
```
nvm use 16
```

# 2. Build local
- Server
```
npm install
npm run dev
npm run watch

npm run run:seeder
```

- Client
```
npm install
npm run start
```
# 3. Build Production
- Server
```
npm install
npm run build
```

- Client
```
npm install
npm run build
```
