# interview-cooking-web
This project is for interview-cooking-web

# 1. Requirement
NodeJs v16

# 2. Build local
- Server
```
npm install
npm run dev
npm run watch
```

- Client
```
npm install
npm run start
```
# 3. Build Production
- Server
```
npm install
npm run build
```

- Client
```
npm install
npm run build
```
